HookLoader

HookLoader は phpBB3 の Hook を管理するための Hook です。 HookLoader を利用すれば管理画面を通して Hook を管理できるようになります。管理可能な Hook は上にタブで表示されます。

Hook System

phpBB3 には Hook System という機能が備わっています。 Hook の意味は 「ひっかける」 です。服を吊るすためのフックをイメージしてもらうと分かりやすいかもしれません。要は、ファイルを置くだけで効果を発揮するシステムが phpBB3 には用意されていてそれを phpBB3 では Hook system と呼んでいる、ということです。具体的には includes/hooks ディレクトリ直下に Hookファイル を置くことでその Hook が機能します。phpBB3 から見れば HookLoader はその Hook の１つに過ぎません。HookLoader ができることは、HookLoader に追加されたプラグインを phpBB3 上で Hook と同じように動作させその Hook を管理画面を通して管理できるようにすることだけです。

HookLoader プラグイン

HookLoader の各プラグインには２つの機能が備わっています。１つは Hook 機能です。組み込み方が違うだけで phpBB3 の Hook と同様に機能します。もう１つは管理機能です。この画面を見てもらえれば分かってもらえるように、管理画面を通してその Hook の動作を管理できます。

プラグインのセットアップ

プラグインを利用するにはプラグインのセットアップが必要です。サイドメニューの “- セットアップ -” からセットアップを行ってください。セットアップを行うまでそのプラグインの Hook 機能は動作せず、管理機能も利用できません。

キャッシュ

サイドメニューの “キャッシュ” からキャッシュを消去できます。
