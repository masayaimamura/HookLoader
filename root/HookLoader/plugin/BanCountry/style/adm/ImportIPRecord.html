IPアドレスレコードのデータベースに関して

BanCountry は PHPパッケージ である IPtoCountry [PHP] を利用しています。
IPtoCountry [PHP] は SQLiteデータベースファイル IPtoCountry.db を利用しています。
IPtoCountry.db は Hakellパッケージ IPtoCountry [Haskell] を利用して作成されています。
よってもし IPtoCountry.db を最新のデータにしたい場合、IPtoCountry [Haskell] を利用できる環境が必要です。
IPtoCountry [Haskell] は UNIX系OS 上で利用できます（32bit版Ubuntu で動作確認済み）。
もし動作しない場合は Haskellコンパイラ(v.7.4.2) で IPtoCountry [Haskell] の Haskellソースコード をコンパイルしてください。
詳しくは Ocean Labo でご質問ください。

IPtoCountry [Haskell] に関して

IPtoCountry [Haskell] は下記の RIR 情報に基づいて SQLiteデータベースファイル IPtoCountry.db を作成しています。
最新の IPtoCountry.db を作成するには下記 RIR 情報をテキストファイル .txt にコピー＆ペーストして IPtoCountry [Haskell] パッケージ直下の RIR ディレクトリに保存してください。
その上でコマンドライン上で
$ ./iptocountry import IPtoCountry.db RIR
と打ち込むことで SQLiteデータベースファイル IPtoCountry.db が作成されます。

RIR(地域インターネットレジストリ)情報

[afrinic] ftp://ftp.afrinic.net/pub/stats/afrinic/delegated-afrinic-latest
[apnic] ftp://ftp.apnic.net/pub/stats/apnic/delegated-apnic-latest
[arin] ftp://ftp.arin.net/pub/stats/arin/delegated-arin-latest
[lacnic] ftp://ftp.lacnic.net/pub/stats/lacnic/delegated-lacnic-latest
[ripencc] ftp://ftp.ripe.net/pub/stats/ripencc/delegated-ripencc-latest
